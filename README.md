2. Install dependencies: npm install

## Usage

**HomePage(index.html)** one can type in the search bar, resluts will display for one to see. If one presses the search button it will redirect one to the **filterpage**

**FilterPage(filter.page)**, one types in the search input, picks a genre and it will dipslay movies cards that are for the that genre and match the input.

**FeedBackPage(feedback.page)**, the user may provide their insights on how to better the wrbiste

**AdvancedFilterPage(adavanced-filter.html)**, one types a movie and is provided the option to filter using year of release, genree and rating.

**FavoritePage(favorite.html)**, one searches for the movie and is capable of saving the movie within the page.
